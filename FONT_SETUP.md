# Geist Font Setup Instructions

## Option 1: Download Geist from Official Source

1. Visit: https://vercel.com/font
2. Download the Geist font family
3. Extract the TTF files
4. Copy these files to `assets/fonts/`:
   - `Geist-Regular.ttf`
   - `Geist-Medium.ttf`
   - `Geist-SemiBold.ttf`
   - `Geist-Bold.ttf`

## Option 2: Use Inter Font (Available via Expo)

Inter is very similar to Geist and readily available through Expo.

Run:
```bash
npx expo install expo-font @expo-google-fonts/inter
```

Then the font will be automatically available.
